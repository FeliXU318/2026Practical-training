# 后端接口文档

## 统一响应格式

成功：

```json
{
  "success": true,
  "message": "操作成功",
  "data": {}
}
```

失败：

```json
{
  "success": false,
  "message": "失败原因",
  "data": null
}
```

## HTTP 状态码说明

| 状态码 | 说明 |
| --- | --- |
| 200 | 请求成功 |
| 201 | 创建成功 |
| 400 | 请求参数错误 |
| 401 | 未登录或 token 失效 |
| 403 | 无权限 |
| 404 | 资源不存在 |
| 422 | 参数校验失败 |
| 500 | 服务器内部错误 |

## 后端建议表结构

### `admin_user` 管理员表

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | bigint | 主键 |
| username | varchar(64) | 登录账号 |
| password_hash | varchar(255) | 密码哈希 |
| role_name | varchar(64) | 角色名称 |
| status | tinyint | 状态 |
| created_at | datetime | 创建时间 |

### `product` 商品表

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | bigint | 主键 |
| code | varchar(64) | 商品编号 |
| name | varchar(128) | 商品名称 |
| type | varchar(32) | 商品类型 |
| merchant_name | varchar(128) | 商家名称 |
| price | decimal(10,2) | 价格 |
| stock | int | 库存 |
| audit_status | varchar(32) | 审核状态 |
| reject_reason | varchar(255) | 驳回原因 |
| submitted_at | datetime | 提交时间 |

### `secondhand_item` 二手商品表
