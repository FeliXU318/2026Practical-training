# 社交新零售电商平台后台管理员网页端

面向校园综合生活服务电商平台的后台运营管理前端项目，支持数据看板、商品审核、二手审核、订单管理、配送管理、电话卡配置、活动公告和投诉举报处理。

## 技术栈

- Vue 3 + Vite
- JavaScript
- Vue Router
- Pinia
- Axios
- Element Plus
- ECharts
- xlsx
- Composition API + `<script setup>`

## 安装依赖

```bash
npm install
```

## 启动开发环境

```bash
npm run dev
```

## 打包构建

```bash
npm run build
```

## 默认登录账号

- 账号：admin
- 密码：123456

## 页面功能列表

- 登录：管理员账号登录、token 和管理员信息本地保存
- 数据看板：统计卡片、订单趋势、业务销量、订单状态分布、待办事项
- 商品审核：查询、查看详情、通过、驳回并填写原因
- 二手审核：查询、查看图片详情、通过、驳回并填写原因
- 订单管理：订单查询、详情查看、订单状态修改
- 配送管理：配送任务查询、分配配送员、更新状态、标记异常
- 电话卡管理：办理入口配置、二维码预览、启用停用
- 活动公告：活动和公告新增、编辑、启用、停用、删除
- 投诉举报：投诉查询、详情查看、填写处理结果
- Excel 导出：支持商品、二手、订单、配送、活动、公告和投诉列表导出
